# Calorie_Checker
Decision-making Fuzzy Inference System (FIS) for checking calorie count.

Instructions:

Open .fis file:
•	Download .fis file
•	Open MATLAB 
•	Click ‘Open’
•	Select .fis file and click ‘Open’

Open Fuzzy Controller:
•	Enter the command ‘fuzzy’ in the Command Window
•	Click ‘File’ on Fuzzy Logic Designer
•	‘Import’ –> ‘From File…’
•	Select file and click ‘Open’
•	Click ‘File’ on Fuzzy Logic Designer
•	‘Export’ –> ‘To Workspace…’
•	Click ‘Ok’
